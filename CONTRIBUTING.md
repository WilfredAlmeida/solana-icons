# Contributing

## Icon Guidelines

### Design Specs
- **Size**: 24x24px viewBox
- **Stroke**: 2px default
- **Style**: Minimal, flat, recognizable
- **Format**: Clean SVG with optimized paths

### Naming Convention
- Lowercase, hyphenated
- Format: `{category}/{project-name}.svg`
- Examples: `wallets/phantom.svg`, `dexes/jupiter.svg`

### Submission Process
1. Add SVG to appropriate category folder
2. Ensure it follows design specs
3. Optimize SVG (remove unnecessary attributes)
4. Create PR with clear description
5. Include source/attribution if applicable

## License
All icons must be MIT compatible or original work. Include attribution in PR if required.
